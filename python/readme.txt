Lai pythone scraper scripts strādātu vajag izveidot vituālo mašīnu iekšā failā.
Vienkārši seko setup, svarīgi lai ir ielādēts python.

Setup:
    1. Run iekšā /python:
        python -m venv venv
            (Windows) venv\Scripts\activate
            (Linux) source venv/bin/activate
        pip install -r requirements.txt
    2. Test run:
        python scripts/scraper.py
